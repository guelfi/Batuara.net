Análise da Solução Batuara.net

Visão Geral

A solução Batuara.net é uma aplicação web moderna construída com .NET 8 e ASP.NET Core. Ela segue uma arquitetura limpa (Clean Architecture), separando as responsabilidades em diferentes projetos para promover a manutenibilidade e a testabilidade.

Arquitetura

A solução é dividida nos seguintes projetos:

  • Batuara.Domain: Contém as entidades de negócio, a lógica de domínio e as regras de negócio principais.
  • Batuara.Application: Orquestra a lógica de negócio, utilizando padrões como MediatR para o desacoplamento de componentes e AutoMapper para o mapeamento de objetos.
  • Batuara.Infrastructure: Lida com as preocupações de infraestrutura, como o acesso a dados com o Entity Framework Core e o logging com o Serilog.
  • Batuara.API: Expõe a funcionalidade da aplicação como uma API web RESTful, utilizando o Swashbuckle para a documentação da API (Swagger).

Tecnologias

  • Framework: .NET 8
  • API: ASP.NET Core
  • Banco de Dados: PostgreSQL
  • ORM: Entity Framework Core
  • Logging: Serilog
  • Documentação da API: Swashbuckle (Swagger)
  • Padrões: MediatR, AutoMapper, FluentValidation

Funcionalidades

A análise das entidades de domínio revela as seguintes funcionalidades principais:

  • Gerenciamento de Calendário:
    • Agendamento de atendimentos (Kardecista, Umbanda, Palestras, Cursos).
    • Controle de capacidade e necessidade de inscrição.
  • Gerenciamento de Eventos:
    • Criação e gerenciamento de eventos (festas, celebrações, bazares).
    • Associação de datas, locais e imagens aos eventos.
  • Conteúdo Espiritual:
    • Gerenciamento de informações sobre Orixás.
    • Gerenciamento de informações sobre as linhas de Umbanda.
    • Publicação de conteúdo espiritual (orações, ensinamentos, hinos, etc.).
