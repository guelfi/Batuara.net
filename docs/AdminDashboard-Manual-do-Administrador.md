# Manual do Administrador — AdminDashboard Batuara

## Acesso

1. Abra o AdminDashboard
2. Faça login com um usuário com perfil `Admin` ou `Editor`
3. Use o menu lateral para acessar os módulos

## Nossa História

Use esta tela para atualizar a apresentação institucional da casa.

### O que pode ser feito

- editar título e subtítulo
- editar a missão da casa
- editar conteúdo rico com:
  - negrito
  - itálico
  - listas
  - citações
- salvar diretamente o conteúdo textual consolidado

### Boas práticas

- mantenha textos institucionais padronizados
- valide o texto final no PublicWebsite após salvar
- use o botão Recarregar para recuperar o conteúdo persistido

## Guias e Entidades

### Operações disponíveis

### Mensalidades

Cada linha de mensalidade permite registrar:

- mês de referência
- vencimento
- valor

### Recomendação operacional

- mantenha uma mensalidade por mês de referência
- ao marcar como pago, preencha também a data de pagamento

## Doações e Contato

Essa tela reúne duas áreas operacionais.

### Contato institucional

Atualize:

- e-mail institucional
- telefone principal
- telefone secundário
- WhatsApp
- horário de atendimento

### Doações e PIX

Atualize:

- chave PIX
- favorecido
- cidade
- banco
- agência
- conta
- tipo de conta
- CNPJ

Após salvar, o PublicWebsite passa a exibir os dados atualizados e o QR Code PIX correspondente.

### Mensagens recebidas

A grade inferior exibe as mensagens enviadas pelo formulário público.

Para tratar uma mensagem:

1. abra o item
2. revise os dados do remetente
3. altere o status
4. registre notas administrativas
5. salve o atendimento

### Status disponíveis

- Nova
- Em atendimento
- Resolvida
- Arquivada

## Reflexo no PublicWebsite

As alterações impactam diretamente:

- seção Sobre
- seção Doações
- seção Contato
- seção Localização
- rodapé

Além disso:

- Eventos mostram apenas o mês corrente
- Calendário mostra apenas o mês corrente
- mensagens do formulário são persistidas no banco

## Cuidados operacionais

- valide URLs antes de salvar
- use textos institucionais revisados
- evite duplicidade de cadastros
- mantenha status corretos para não exibir conteúdo inadequado
- confira o site público após mudanças críticas
